    • Имя пользователя и e-mail

    используются для коммитов
    добавляем в глобальные настройки:
    • git config --global user.name "myname"
    • git config --global user.email myemail@mymail.com
    если для конкретного проекта требуются другие имя и e-mail, то надо устеновить их без --global

    Проверка настроек

    • git config --list - все настойки
    • git config user.name - конкретная настройка

    Получение справки

    • git help <команда>

    Инициализация репозитария

    надо перейти в требуюмую папку и ввести:
    • git init

    Клонирование существующего репозитария

    • git clone https://github.com/mvalnew/r1 myFolder

    Проверка состояния файлов

    • git status
    • git status -s - сокращенный вывод

    Начало слежения за новыми файлами и принятие изменений

    • git add myFile.txt

    Игнорирование файлов

    Необходимо создать файл .gitignore и поместить в рабочую область. Список паттернов:
    • *.[oa] - с расширением .o или .a
    • *~ - заканчиваются на ~

    Фиксация изменений

    • git commit -m "my comment"

    Удаление файлов

    • git rm myFile.txt

    Дополнение последней фиксации

    • git commit --amend

    Отмена индексирования

    • git reset HEAD myFile.txt

    Отмена изменений в файле

    • git checkout -- myFile.txt

    Отображение удаленных репозитариев

    • git remote -v

    Добавление удаленных репозиториев

    • git remote add myRepo https://github.com/mvalnew/r1

    Скачивание из удаленных репозиториев

    • git fetch [myRepo]
    • git pull [myRepo] // скачивание и слияние

    Отправка данных в удаленный репозитарий

    • git push myRepo master
