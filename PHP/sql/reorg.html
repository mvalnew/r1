Преобразования в операторе ALTER
СинтаксисОписание
ADD [COLUMN] описание_столбца [FIRST | AFTER столбец]добавление столбца в указанное место
ADD [COLUMN] (описание_столбца, описание_столбца, ...)добавление столбцов
ADD INDEX [индекс] (столбец, столбец, ...)добавление индекса
ADD [CONSTRAINT [символ]] PRIMARY KEY (столбец, ...)Создание первичного ключа
ADD UNIQUE [CONSTRAINT [символ]] [индекс] (столбец, ...)Добавление уникального индекса
ADD [CONSTRAINT [символ]] FOREGIN KEY [индекс] (индексный_столбец, ...) [определение_ссылки]добавление внешнего ключа
ALTER [COLUMN] {SET DEFAULT значение | DROP DEFAULT}добавление или удаление значения по умолчанию для столбца
CHANGE [COLUMN] столбец новое_описание_столбцаизменение столбца, в т.ч. имени
MODIFY [COLUMN] описание столбцаизменение типа столбца, но не имени
DROP [COLUMN] столбецудаление столбца
DROP PRIMARY KEYудаление первичного индекса (но не столбца)
DROP INDEX индексудаление индекса
DROP FOREGIN KEY ключудаление внешнего ключа (но не столбца)
DISABLE KEYSотключение обновления индексов
ENABLE KEYSвключение обновления индексов
RENAME [AS] новое_имя_таблицыпереименование таблицы
ORDER BYповторно создает таблицу с упорядочиванием строк (порядок не сохраняется при изменении в таблице)
CONVERT TO CHARACTER SET набор_символов COLLATE сопостпреобразование всех тестовых столбцов к указанному набору символов и правилам сопоставления
[DEFAULT] CHARACTER SET набор_симв COLLATE сопостустановка набора символов и правил сопоставления по умолчанию
DISCARD TABLESPACEудаление лежащего в основе файла табличной памяти
IMPORT TABLESPACEповторное создание лежащего в основе файла табличной памяти
парметры таблицыпереустановка параметров таблицы

пример : 
alter table orders
add tax float(6,2) after amount
