    • Файлы

    Открытие файла

    • $fp = fopen ("$DOCUMENT_ROOT/../myfile.txt", 'w'); // false если неудача, $SERVER['DOCUMENT_ROOT'] - корневой каталог
      • r - чтение
      • r+ - чтение и запись с начала файла
      • w - запись с начала
      • w+ - запись и чтение
      • x - осторожная запись (если файла еще нет)
      • x+ - осторожная запись и чтение
      • a - добавление
      • a+ - добавление и чтение
      • b - бинарный режим (по умолчанию)
      • t - текстовый режим

    Запись в файл

    • int fwrite(resource handle, string string [, int length])
    • int file_put_contents(string filename, string data) - отрывает, записывает и закрывает файл
    • Есть связанная функция file_get_contents()

    Закрытие файла

    • fclose($fp)

    Чтение всего файла

    • $filearray = file ("$DOCUMENT_ROOT/../myfile.txt"); // считывание в массив строк
