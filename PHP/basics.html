    Доступ к переменным формы

    • $_POST['myname']
    • $_GET['...']
    • $_REQUEST['...']

    Конкатенация

    • echo $myvar.' ...';
    • echo "$myvar ..."; - в двойных кавычках

    Типы

    • Integer
    • Float
    • String
    • Bolean
    • Array
    • Object
    • NULL
    • resource

    Приведение типов

    • $var = (float)$var2;

    Переменные переменных

    • $varname = "var";
    • $$varname = 5; то же, что и $var = 5;

    Константы

    Можно увидеть в phpinfo()
    • define('MYCONST', 100);
    • ...
    • echo MYCONST; - без $

    Суперглобальные переменные

    • $_GET - массив переменных переданных методом GET
    • $_POST - массив переменных переданных методом POST
    • $_COOKIE - массив cookie-переменных
    • $_FILES - массив переменных, относящихся к загрузке файлов
    • $_ENV - массив переменных окружения
    • $_REQUEST - массив переменных включающий $_GET, $_POST, $_COOKIE
    • $_SESSION - массив переменных сеанса

    Арифм. операции

    • +, -, *, /, %

    Строковые операции

    • $result = $a.$b;

    Присвоение

    • $a = $b = $c;
    • +=, -=, *=, /=, %=, .=
    • ++, --

    Ссылки

    • $a = 5;
    • $b = &$a;
    • $b = 7; - $a и $b равны
    • unset($a); - сброс $a, $b по прежнему равно 7

    Сравнения

    • == равно
    • === идентично
    • != не равно
    • !== не идентично
    • <> не равно
    • <, >, <=, >=

    Логика

    • !, &&, ||
    • and, or, xor с меньшим приритетом

    Тернарная операция

    • условие ? если_истина : если_ложь

    Подавление ошибок

    • $a = @(57 / 0);

    Операция выполнения

    • $out = `dir`; // в обратных кавычках
    • echo "<pre>$out</pre>";

    Операции с массивами

    • + - объединение
    • == - равно (одни и те же элементы)
    • === - идентично (одни и те же элементы в том же порядке)
    • !=, <> - не равно
    • !== - не идентично

    Операция определения типа

    • if ($myObject instanceof myClass) ...

    Получение и установка типа переменной

    • string gettype(mixed var); - bool, int, double, string, array, object, resource, NULL
    • int settype(string var, string type);
    • is_array(), is_double(), is_int(), is_string(), is_bool(), is_object(), is_resource(), is_null(), is_scalar(), is_numeric(), is_callable()

    Получение и установка состояния переменных

    • bool isset(mixed var [, mixed var , ...])
    • void unset(mixed var [, mixed var , ...])
    • int empty(mixed var)

    Изменение типа переменной

    • int intval(mixed var);
    • float floatval(mixed var);
    • string strval(mixed var);

    Управляющие структуры

    • if (...) { ...} elseif(..) { ... } else { ... }
    • switch (...) {case ...: break; default: break;}
    • while (...) { ... }
    • for (...; ...; ...) { ... }
    • do { ... } while (...)
    • break, continue
    • if (...): ... elseif(..): ... else: ... - альтернативный синтаксис
