• $a + $b - Массив $a добавляется к $b, элементы с конфликтующими ключами не добавляются
  • $a == $b - равно (одниаковые элементы)
  • $a === $b - идентично (однинаковые элементы с одинаковым типом в том же порядке)
  • $a != $b, $a <> $b - не равно
  • $a !== $b - не идентично
  • array_push($myarray, $el) - добавление в конец массива
  • count($myarray) - количество элементов
  • array_count_values($myarray) - возвращает ассоциативный массив с частотами элементов
  • $myarray = explode(сепаратор, $mystring, $maximum) - разбиение строки
  • $el = current($myarray) - текущий элемент
  • $el = each($myarray) - текущий элемент + шаг вперед
  • $el = next($myarray) - шаг вперед + текущий элемент
  • $el = prev($myarray) - обратная next
  • $el = reset($myarray) - в начало + первый элемент
  • $el = end($myarray) - в конец + последний элемент
  • extract($myarray, [extract_type, prefix]) - разложение на переменные с именами ключей
    • EXTR_OVERWRITE - если конфликт перезапись (по умолчанию)
    • EXTR_SKIP - если конфликт пропуск
    • EXTR_PREFIX_SAME - если конфликт добавляет префикс
    • EXTR_PREFIX_ALL - всегда добавляет префикс (по умолчанию)
    • EXTR_PREFIX_INVALID - добавляет префикс для недопустимых имен
    • EXTR_IF_EXISTS - извлечение только существующих переменных
    • EXTR_PREF_IF_EXIST - добавляет префикс если без префикса уже существует
    • EXTR_REFS - извлекает переменные как ссылки
