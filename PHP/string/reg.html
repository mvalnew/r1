Два вида синтаксиса : POSIX и Perl
Рассматриваем POSIX :
  • "." - одиночный символ (кроме перевода строки)
  • [а-я] [аеиоуыэюя] [a-zA-Z] - один из символов набора
  • [^аеиоуыэюя] - не из набора
  • Предопределенные классы символов :
    • [[:alnum:]] - алфавитно-цифровые
    • [[:alpha:]] - буквы
    • [[:lower:]] - строчные буквы
    • [[:upper:]] - прописные буквы
    • [[:digit:]] - десятичные цифры
    • [[:xdigit:]] - шестнадцеричные цифры
    • [[:punct:]] - пунктуация
    • [[:blank:]] - табуляции и пробелы
    • [[:space:]] - любые пробельные символы
    • [[:cntrl:]] - управляющие символы
    • [[:print:]] - печатные символы
    • [[:graph:]] - печатные кроме пробельных
  • +, * - одно и больше повторений, 0 и больше
  • (очень )*крутой - скобки указывают подвыражение
  • (очень ){1, 3} - подвыражение с количеством повторений (если не указано окончание и запятая, то точное значение, если запятая есть то количество повторений ограничено только снизу)
  • ^, $ - в начале и в конце
  • com|edu|net - ветвление (одно из)
  • Спец.символы - предваряются \
    \ ^ $ . | ( ) * + { } ? (за пределами квадратных скобок)
    \ ^ - (внутри квадратных скобок)
  • int ereg(string рег_выражение, string где_ищем [, array массив_соответствий]) - поиск соответствий (возвращает длину совпавшей строки или false)
  • ieregi() - без учета регистра
  • int ereg_replace(string рег_выражение, string на_что_мняем, string где_ищем) - поиск и замена соответствий (устарело - надо заменить на preg_replace)
  • eregi_replace() - без учета регистра
  • array split(string рег_выражение, string где_ищем [, int макс_колич]) - разбиение
